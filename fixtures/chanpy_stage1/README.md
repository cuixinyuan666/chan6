# chan.py stage-1 fixtures

This directory is for strict alignment between Chan6 Rust stage-1 output and hichan `chan.py` output.

## Files

```text
input/stage1_small.csv
```

Small deterministic K-line input for the first alignment pass.

Gold output is intentionally **not committed yet** until it is generated by the real hichan `chan.py` engine:

```text
gold/stage1_small_chanpy_gold.json
```

## Generate gold output

From the Chan6 repository root, run:

```bash
python tools/generate_chanpy_stage1_gold.py \
  --hichan-repo ../chan_replay_app \
  --input fixtures/chanpy_stage1/input/stage1_small.csv \
  --out fixtures/chanpy_stage1/gold/stage1_small_chanpy_gold.json
```

Or set an environment variable:

```bash
export HICHAN_REPO=../chan_replay_app
python tools/generate_chanpy_stage1_gold.py
```

On Windows Git Bash:

```bash
HICHAN_REPO="../chan_replay_app" python tools/generate_chanpy_stage1_gold.py
```

## Rule

Do not hand-write `gold/stage1_small_chanpy_gold.json`.

The gold file must come from:

```text
cuixinyuan666/chan_replay_app
branch: hichan
backend/app/chanpy_engine.py::analyze_bars
```

Only after the real gold JSON exists should Rust include/fx/bi be changed to match it.
